# React Dashboard & Payment App

Bu loyiha React yordamida yaratilgan oddiy **Dashboard va Payment** tizimi. U foydalanuvchilarni boshqarish, to‘lovlarni kiritish va hisob ma’lumotlarini ko‘rish imkonini beradi. Loyihada quyidagi sahifalar va komponentlar mavjud:

## Sahifalar
- **Home.js** – Bosh sahifa, `AdminPanel` va `PaymentForm` komponentlarini ko‘rsatadi.
- **Login.js** – Foydalanuvchi login sahifasi.
- **Dashboard.js** – Admin va Payment boshqaruv paneli.
- **AccountDetail.js** – Foydalanuvchi hisob tafsilotlari sahifasi.

## Komponentlar
- **AdminPanel.js** – Foydalanuvchilarni qo‘shish va boshqarish formasi.
- **PaymentForm.js** – To‘lov kiritish formasi.

## Texnologiyalar
- React.js
- React Hooks (`useState`, `useEffect`)
- Inline styling (CSS-in-JS)
- JavaScript ES6+
